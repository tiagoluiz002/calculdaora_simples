# Calculadora em Java

Projeto simples de calculadora desenvolvido em Java, criado como parte do meu aprendizado da linguagem.

## 💡 Sobre o projeto

Esta calculadora permite realizar as quatro operações matemáticas básicas (soma, subtração, multiplicação e divisão) através do terminal.

## 🚀 Funcionalidades

- Leitura de dois números via teclado
- Escolha da operação desejada (+, -, *, /)
- Tratamento de erro para divisão por zero
- Validação de operação inválida

## 🛠️ Tecnologias utilizadas

- Java

## ▶️ Como executar

```bash
javac Main.java
java Main
```

## 📚 Objetivo

Projeto criado com fins de aprendizado, praticando conceitos como:
- Variáveis e tipos primitivos
- Estruturas condicionais (`if/else`)
- Leitura de entrada com `Scanner`
- Comparação de Strings com `.equals()`
